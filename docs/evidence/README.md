# Evidence Folder

This folder is reserved for real evidence captured after approved deployment or account actions.

Do not add placeholder screenshots or fabricated URLs. When evidence exists, add files with descriptive names such as:

- `ai-studio-project-overview-YYYY-MM-DD.png`
- `cloud-run-service-YYYY-MM-DD.png`
- `mobile-smoke-test-YYYY-MM-DD.png`
- `desktop-smoke-test-YYYY-MM-DD.png`

After adding evidence, update `docs/FINAL_EVIDENCE_REPORT.md` with the exact filename, capture date, environment, and what the screenshot proves.
